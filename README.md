
# 用 Vue Router 實作網站頁面切換
> Vue Router 是 Vue.js 官方提供的路由管理工具。<br>
> 此專案練習以 **Travels** 為主題，實作籤套路由、返回上一頁、404頁面等功能。<br>

🚀 &nbsp; 快速開始：<br>
- 安裝相依套件
```
npm install
```
- 啟動開發伺服器
```
npm run serve
```
